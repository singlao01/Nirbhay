# Nirbhay - Women's Safety App

## 🚀 About Nirbhay
**Nirbhay** is a **women's safety application** designed to provide **quick emergency assistance, live location tracking, and safety tips**. It enables users to send emergency alerts, access essential helpline contacts, and stay informed about women's safety.

## 🌟 Features
- 🚨 **Emergency Alert** - Notify trusted contacts with your live location.
- 📍 **Live Location Sharing** - Share your real-time location when traveling alone.
- 📞 **Emergency Contacts** - Quick access to helpline numbers and personal emergency contacts.
- 📰 **Live News Updates** - Stay informed about women's safety news via an integrated news API.
- 🔒 **Login System** - Secure authentication for users.
- 🛡️ **Safety Tips** - Access self-defense techniques and personal safety guidelines.


## 🛠️ Tech Stack
- **Frontend**: React.js, React Router, Axios
- **Backend**: Node.js, Express.js, MongoDB
- **Styling**: CSS, Responsive UI
- **API Integration**: NewsAPI for real-time safety news

## 🎯 Installation & Setup
### Clone the Repository
```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/Nirbhay.git
cd Nirbhay
```

### Install Dependencies
```bash
npm install
```

### Run the Frontend
```bash
cd frontend
npm start
```

### Run the Backend
```bash
cd backend
node server.js
```

## 🌍 API Configuration
1. Create a `.env` file in the backend directory.
2. Add the following variables:
```env
MONGO_URI=your_mongodb_connection_string
NEWS_API_KEY=your_news_api_key
```
